      I've been using this theme for years now. It's user friendly and does the job! One note that's come up recently - it would be really helpful if a customer had to select their variant before being able to add to cart. Right now, a variant is selected by default, so if a customer is just quickly shopping they might miss that there are options to choose from. Not a big deal, just a small UX detail!
